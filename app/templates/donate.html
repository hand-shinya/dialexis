Dialexis is an open (AGPL-3.0) philosophy research infrastructure standing on free, keyless public scholarly foundations (Wikidata, OpenAlex, Crossref, OpenCitations, Wikisource…). Browsing and the full research desk are free, permanently.

Only three things cost money:

  • Server hosting (a ~$15/month-class VPS)
  • A future community LLM pool (shared AI compute for students and users without API keys — designed, not yet live)
  • Translations, teaching materials, expert review (future)

Ways to support (some in preparation):

  • GitHub Sponsors: github.com/sponsors/hand-shinya (being set up)
  • Contribute code, translation data, or reading-level texts: GitHub
  • Using your own API key is itself a contribution: it scales usage at zero cost to the commons (Axiom 5)

Income and spending will be published on this page — transparency applies to donations too.
