# COST_AND_API.md — 費用とAPI設計書

原則（GENESIS公理5）：**無料・鍵不要を根幹とし、費用は (a)利用者のBYOキー か (b)透明な運営費 にのみ発生させる。**

## 1. 使用中の外部API（すべて無料・鍵不要）

| API | 用途 | 制限・作法（2026-07時点の理解。変更されうる） |
|---|---|---|
| Wikidata (wbsearchentities / EntityData) | 人物・概念の同定、多言語ラベル、著作・影響関係 | 連絡先つきUser-Agent必須級。キャッシュ24h |
| OpenAlex | 論文・著者・引用数・新着 | 無償。`mailto`付き（polite pool）推奨=環境変数 `DIALEXIS_CONTACT`。日次上限は寛大 |
| Crossref | DOIメタデータ・新着 | 無償。politeアクセス同上 |
| OpenCitations | DOI引用リンク | 無償（トークン任意）。補助的利用 |
| Wikipedia REST (summary) | 概念・人物の要約（多言語） | 無償。キャッシュ24h |
| Wikisource | 原典全文へのリンク | リンクアウトのみ（本文は取り込まない＝公理3） |
| Gutendex (Project Gutenberg) | パブリックドメイン原典 | 無償 |

実装上の遵守：全リクエストは `connectors/base.py` を通り、(1)連絡先つきUA (2)SQLiteキャッシュ (3)explore1回≦7リクエスト、が構造的に保証される。

## 3. LLM費用（利用者BYOキー・運営者負担ゼロ）

- 対応：Anthropic / OpenAI / Google Gemini / Ollama（ローカル・完全無料）
- 鍵はブラウザ localStorage のみ。サーバーは保存・ログせず、リクエスト毎に中継
- 費用感の目安：反証エンジン1回 ≒ 入力1〜2k+出力1〜2kトークン。各社の最廉価モデルなら1回あたり概ね1円未満〜数円（**概算・未検証。各社の現行価格表を確認のこと**）
- 無料で使いたい利用者への公式回答：**Ollama**（手元PCでllama等を動かす）を設定すれば費用ゼロでLevel 2が使える
